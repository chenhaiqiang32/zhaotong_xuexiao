# 门锁 接口文档（jetlinks-community）

## 接口 1：分页查询门锁信息

- **URL**：`POST http://172.255.4.7:8848/api/smart-lock/page`
- **描述**：分页查询门锁信息列表
- **白名单**：是（无需登录）

### 请求参数（Body JSON）

| 字段 | 类型 | 必填 | 说明 |
|---|---|---:|---|
| `pageNum` | number | 否 | 页码，默认 1 |
| `pageSize` | number | 否 | 每页条数，默认 10 |
| `apartmentName` | string | 否 | 楼栋名称（模糊查询） |
| `floorName` | string | 否 | 楼层名称（模糊查询） |
| `roomName` | string | 否 | 房间名称（模糊查询） |
| `status` | number | 否 | 门锁状态（如 1 在线、2 离线等） |
| `lockName` | string | 否 | 门锁名称（模糊查询） |

### 请求示例

```json
{
  "pageNum": 1,
  "pageSize": 12
}
```

### 响应示例（成功）

```json
{
  "message": "success",
  "result": {
    "code": "200",
    "message": "操作成功",
    "data": {
      "total": 72,
      "list": [
        {
          "id": 1005,
          "mac": "80A036AE3805",
          "name": "A慎行楼206锁",
          "apartmentName": "慎行楼",
          "floorName": "2F",
          "roomName": "206室",
          "uuid": "80A036AE3805",
          "status": 1,
          "battery": 100,
          "firmwareVersion": "V20:R6.00.9.52"
        }
      ],
      "pageNum": 1,
      "pageSize": 10,
      "pages": 8
    }
  },
  "status": 200,
  "timestamp": 1776058639604
}
```

### 响应字段说明（核心）

| 路径 | 说明 |
|---|---|
| `status` | HTTP 风格状态码（示例为 200） |
| `message` | 网关/包装层信息（示例为 success） |
| `result.code` | 业务码（成功为 `"200"`） |
| `result.message` | 业务消息 |
| `result.data.total` | 总记录数 |
| `result.data.list` | 列表数据 |
| `result.data.pageNum/pageSize/pages` | 分页信息 |

## 接口 2：分页查询开门记录

- **URL**：`POST http://172.255.4.7:8848/api/smart-lock/pageOpenLog`
- **描述**：分页查询门锁开门记录
- **白名单**：是（无需登录）

### 请求参数（Body JSON）

| 字段 | 类型 | 必填 | 说明 |
|---|---|---:|---|
| `pageNum` | number | 否 | 页码，默认 1 |
| `pageSize` | number | 否 | 每页条数，默认 10（建议传数字） |
| `name` | string | 否 | 姓名（模糊查询） |
| `cardCode` | string | 否 | 卡号（模糊查询） |
| `lockName` | string | 否 | 门锁名称（模糊查询） |
| `apartmentName` | string | 否 | 楼栋名称（模糊查询） |
| `startTime` | string | 否 | 开始时间，格式：`yyyy-MM-dd HH:mm:ss` |
| `endTime` | string | 否 | 结束时间，格式：`yyyy-MM-dd HH:mm:ss` |

### 请求示例

```json
{
  "pageNum": 1,
  "pageSize": 12,
  "lockName": "A敏行楼203锁",
  "apartmentName": "敏行楼"
}
```
